# 🚀 Personal Portfolio & Interactive Showcase

Welcome to my portfolio! I am a results-driven Software Engineer with specialized expertise in integrating Machine Learning models into full-stack applications. This repository contains the source code for my interactive portfolio, featuring a custom MERN stack architecture, dynamic Framer Motion animations, and a seamless "Liquid Glass" UI.

## 🌐 Live Demo

The frontend of this portfolio is actively deployed and running on Vercel. 

👉 **[Click here to view the Live Portfolio](https://portfolio-vert-delta-avo680ue8d.vercel.app/)**

---

## 🛠️ Tech Stack
* **Frontend:** React, Vite, Tailwind CSS, Framer Motion, Lucide Icons
* **Backend:** Node.js, Express.js (RESTful API)
* **Database:** MongoDB (Atlas / Local)
* **Deployment:** Vercel (Frontend)

## 💡 Key Features
* **Interactive Skills Dial:** A custom-built, physics-based UI wheel to explore my technical stack.
* **Sticky Certificate Stacking:** An Apple-style scroll-locked accordion for viewing credentials.
* **Project Carousel:** Interactive video showcases for my monthly Computer Vision and Web-D projects.
* **Direct Contact & Resume:** Integrated backend API to handle messages and dynamically serve my CV.
